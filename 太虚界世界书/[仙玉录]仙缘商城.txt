仙缘商城:
  商品刷新规则:
    1. 境界适配刷新:
       - 刷新商品时，依据宿主的境界，调整不同的商品出现概率和权重。
       - 练气境/凡人：以“凡阶”上中下三品和“黄阶”下品为主。
       - 筑基境：以“黄阶”上中下三品为主，有概率出现“玄阶”中品和下品的物品，极低概率出现“玄阶”上品。
       - 金丹境: 以“玄阶”上品为主，有概率出现“地阶”下品的商品。
       - 元婴境: 以“地阶”下品为主，有概率出现“地阶”中品的商品，极低概率出现“地阶”上品商品。
       - 化神境: 以“地阶”中品为主，有极低概率出现“地阶”上品的商品。
       - 合体境: 以“地阶”上品的商品为主，有概率出现“天阶”下品的商品。
       - 炼虚境: 以“天阶”下品和中品为主，不会出现天阶上品的商品。
       - 大乘境: 以“天阶”上品和“仙品”下品为主，概率出现“仙品”中品和上品
    2.数值设定: 不同品阶商品的数值设定必须严格参考<物品数值基准>
    3. 色情商品激励:
       - 定义：凡是涉及到“性爱”、“调教”等跟男女相关的商品。
       - 受宿主总堕落值影响: 总堕落值越高，刷新出的色情商品越多。
       - 低代价陷阱：色情商品的价格通常比同效能的正统商品更低廉，并且效果、加成都更加夸张。
    4. 固定加成格式与属性限制（刷新生成时必须遵守）:
      统一格式：属性名 + 数值/%，例如“当前灵气+40”“基础悟性+5%”。
        丹药类允许：
          当前属性：当前根骨 / 当前神海 / 当前悟性 / 当前魅力 / 当前气运 / 当前杀伐 / 当前神伤 / 当前横练 / 当前身法 / 当前生命 / 当前灵气 / 当前道心
          基础属性：基础根骨 / 基础神海 / 基础悟性 / 基础魅力 / 基础杀伐 / 基础神伤 / 基础横练 / 基础身法 / 基础生命 / 基础灵气
        其他分类（功法/武器/装备/法宝/着装/阵符/特殊）仅允许：
          当前属性（同上全部当前属性）
          基础生命 / 基础灵气
    5. 补货逻辑:
       - 每当商品被兑换一个，必须立即补充一个新的商品。

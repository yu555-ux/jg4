<修士境界体系>
太虚界_修行境界体系:
  基本概念:
    定义: 太虚界修士通过吸收天地灵气、感悟天道法则而不断提升自身实力的修行体系。

  境界划分:
    凡人:
      数值映射: 0
      实力描述: 未入道途，无法感知天地灵气。肉身脆弱，受生老病死约束，寿元不过百载。无法施展任何法术，只能依靠凡间武学或外力。

    练气境:
      映射节点: 1(一层) / 2(二层) / 3(三层) / 4(四层) / 5(五层) / 6(六层) / 7(七层) / 8(八层) / 9(九层)
      实力描述: 修行之始，修士开始感应并引动天地灵气进入经脉，洗涤肉身。可施展基础法术，使用符箓，短距离御物。寿元延长至百岁左右。

    筑基境:
      映射节点: 10(初期) / 13(中期) / 16(后期) / 19(圆满)
      实力描述: 筑就大道之基，重塑经脉以承载更多更强的灵气。筑基后，修士可长时间御剑飞行，神识初生，内视自身，寿元延长至两百岁左右。

    金丹境:
      映射节点: 20(初期) / 23(中期) / 26(后期) / 29(圆满)
      实力描述: 灵气于体内丹田处凝丹，称为“金丹”，生命本质飞跃。寿元五百载，可施展大范围五行神通，可对天道有模糊感知。

    元婴境:
      映射节点: 30(初期) / 33(中期) / 36(后期) / 39(圆满)
      实力描述: 破丹生婴，孕育第二生命，神魂与灵气高度结合。寿元一千载，金丹毁而元婴存，可在短距离内瞬间移动，神识覆盖百里，可探查周围环境。

    化神境:
      映射节点: 40(初期) / 43(中期) / 46(后期) / 49(圆满)
      实力描述: 灵气化神，元婴升华为元神，灵气开始带上天道法则的属性。在自身领域内对低阶修士拥有绝对的灵气支配权，可压制低阶修士的灵气运转，寿元延长至两千岁左右。

    合体境:
      映射节点: 50(初期) / 53(中期) / 56(后期) / 59(圆满)
      实力描述: 道体合一，肉身、元神与感悟的天道法则彻底融合。拥有近乎不灭的再生能力，即使受到重创也能快速恢复，能够直接撕裂虚空，横渡小世界，在虚空中穿梭，寿元延长至五千岁左右。

    炼虚境:
      映射节点: 60(初期) / 63(中期) / 66(后期) / 69(圆满)
      实力描述: 炼就太虚之身，修士形体融入虚空，灵气由实转虚，可虚空造物，言出法随，在凡间被称为"地仙"，因果感应极其敏锐，能够感知自身与他人的因果联系，寿元延长至万岁。

    大乘境:
      映射节点: 70(初期) / 73(中期) / 76(后期) / 79(圆满)
      实力描述: 凡道之巅，灵气开始向"仙灵之气"转化，肉身与神魂达到凡间承载极限。半步真神，对天道法则有深刻理解。此阶段修士多闭关潜修，等待成仙大劫，一旦渡过便可飞升仙界，寿元接近无限。

  特殊定义:
    瓶颈: 只有角色的某个小境界/大境界的修行进度达到上限，才可以被称为瓶颈。
</修士境界体系>

一、引言  
随着 AI 能力的发展以及使用门槛的降低，预设的研究方向逐渐从“突破限制”“防止截断”转向追求更高质量的输出。例如，提升创作逻辑、追求特定的文风效果等。  
然而，目前仍然没有一份完整的文风教程来指导大家如何创作属于自己的文风。我作为长期研究类脑文风的老玩家，在此分享一些经验，希望能帮助大家学会如何撰写文风。  
本文将介绍通用的文风提示词创作方法，并结合实际案例进行讲解。

---

二、理论基础  
**1. 提示词**  
首先明确一点：要写文风，必须先会写提示词。  
提示词并不难写，是人人都能掌握的技能。这里我先简单介绍一下提示词的基本写法。  

- **明确目标**：你必须清楚自己想要什么效果，这是撰写提示词的前提。  
- **正面表达**：多写“要什么”，少写“不要什么”。直接告诉模型“我要 XX”，比说“我不要 XX”更有效。  
这好比有 10 条路从 A 到 B，如果你告诉司机“走第五条路”，显然比说“别走第一条路”更容易达成目标。  
- **测试与迭代**：写完提示词后进行测试。如果效果不符合预期，就继续补充或修改提示词。如果堆砌过多、难以修改，甚至可以推倒重写，重新整理框架。

**简单示例**  
```
目标：让模型扮演一只猫娘  
提示词：你是一只猫娘，我是你的朋友，让我们聊聊天吧。
```
测试后发现猫娘的性格、外貌、喜好等不符合期望，于是进一步修改：  
```
目标：让模型扮演一只猫娘，并加入个人偏好  
提示词：你是一只傲娇的蓝发猫娘，长相可爱，脸圆圆的，喜欢吹喇叭。你的名字是阿灵，我是你的朋友，让我们聊聊天吧。
```
复杂的提示词也是如此逐步添加和调整而成的。不必畏惧，先动手写出来，再慢慢优化。

如果你在表达上有困难，可以借助 AI 辅助：将你期望的效果尽可能完整地描述出来，让 AI 帮你整理。描述越清晰越好，若不清晰，AI 通常也能理解大致意图。这是一个锻炼表达的好方法，熟练后便可脱离辅助，独立撰写。

但请注意：**建议尽量手写提示词**。如果效果分为三个等级，手写提示词天生就在二级以上，而 AI 生成的提示词即使经过人工精修，最多也只能达到二级。因此，为什么不直接手写呢？  
手写并不难，就像平时说话一样自然。举个例子，把 AI 看作卖早点的，你的话就是 prompt（提示词）。“我想要两个包子一杯豆浆”很简单吧？写提示词也是如此，要什么就写什么。初期不必追求复杂的指令格式，用自己说话的方式写就可以。

**2. 专业知识**  
文风创作并非零基础就能完成，你需要掌握一些基础的文学理论。  
如果你已经具备相关理论知识，那很好，你可以凭借经验快速搭建文风初稿，再进行长期打磨。  
如果完全不了解，也可以通过知乎、B 站、书籍或论文进行学习。系统性学习或碎片化积累均可，首要目标是快速掌握基础理论并加以运用。这种方法虽属“速成”，上限可能不如系统学习，但足以做出效果不错的文风——我自己就是这样起步的。

**3. 文风的两个组成部分**  
我个人从叙事学入门，快速浏览《叙事学导论》后对文风有了新的理解；而 hyh 则是从文体学与论文入手。两者各有侧重，但都重要。我将文风分为两部分：

- **文字结构**：指文字的叙述方式，如叙述人称、聚焦视角、讲述与展示的处理等。  
- **文字感受**：指文本带给读者的体验，如氛围是沉重的、情感是细腻的、文字是准确清晰的等。

若只注重结构，文本可能流于平庸；若只注重感受，则容易陷入“八股”套路。  
两者需结合使用，比例根据个人需求调整。但切记：**尽量手写**。AI 生成的文字容易导致过拟合与八股问题，若想避免，就应亲自撰写。

**4. 针对 AI 写作的一些问题**  
在正式讲解文风前，需要了解当前模型的常见缺陷及其弥补方法。  
不同模型有不同弱点，例如 Gemini 喜欢过度解释和滥用比喻。这些可统称为“八股”，消除它们能在很大程度上改善输出质量。

三、文风提示词的三层框架搭建  
撰写文风与撰写人设类似，首先要**拆解特征**，逐一解释，让模型清楚理解你想要的文风。  
就好比让模型生成“一只狗”，它可能输出拉布拉多，但你想要的是牧羊犬。拆解特征后，模型才能稳定生成符合期待的牧羊犬。

基于长期实践，我总结出一个通用框架：**文风 = 基调 + 文体特征 + 叙述规则**  
三者不必俱全，可根据需要组合使用。

**1. 基调**  
基调相当于主提示词，是文风最核心的审美追求与创作基础，可放置高层次概念或通用要求。  
我个人的基调强调第一人称主观体验、独白写法，以及在强调主观描写的同时避免八股。  
```
<fiction_style>
核心原则：
- 主观真实论：外部世界仅为表象，真实仅存在于个体主观体验之中
- 独白是叙事主体，融入全文
- 所有文字基于一个既定角色的叙述立场，纯粹呈现外部现实和内心世界的原始事实，没有作者介入，人物主观感受到的“真实”再现于文字上
- 尊重读者智力：默认读者理所当然地能读懂文本，不需要替代读者对文本进行上帝视角的解读阐释
具体方法：
- 体验：叙事距离极近，读者能够亲身体验叙述者的想法、感受和动机。
- 有限叙述：使用限知视角，没有人是全知的，角色只能知道自己该知道的
- 行动和对白只呈现，不修饰状态。避免描述非叙述者角色言行神态背后的动机或内涵，将评价权完全交给读者。
- 深入描述角色明显直接的言行举止；忽略不易察觉的任何细节，如角色的细微神态、小动作或其他反应。
- 避免直接给结论的形容词或副词/用概略性的语言一笔带过
- 内心独白：
  - 是主体表达自我的关键手段，表达他的一切思想情感。不分析解读其他人物，不分析猜测别人的动作/心理，只是当下的反应
  - 在主观立场上进行无时空无逻辑非理性非线性极度个人的混乱独白
  - 杜绝对心理状态的总结概述，独白必须是未经整理的即时反应
- 外部现实是引子，重点是紧随其后的内心世界独白呈现
- 叙述者是亲历者，不是旁观者。
</fiction_style>
```
当然我主要是用来杀八股，这个不是什么很牛逼的东西。

**2. 文体特征**  
文体特征指文字带给读者的感受，是期望文风实现的特定效果集合，与文体学相关，关注审美价值与读者体验。  
以下是我曾为网文撰写的文体特征示例：  
```
- 文体特征:
  - 通俗化:
    - 真情实感，辞意妙合:遣词造句追求直接准确，不求精致。要能够清晰表达自己的意思和情感给别人的东西。
    - 文字就是生活状态，写出人物第一时间的真实感受，不会为文而文，没话找话。 
    - 故事的叙述节奏快，追求及时反馈
  - 去严肃化: 
    - 坚持个人的欲望表达，满足创作和读者的自我需求。着眼于叙事但不关心叙事所传达的意义。
    - 拒斥“文以载道”的责任，通过幻想实现自我压力释放的需求，让读者得到娱乐和放松。
    - 故事性: 放弃所指层面的深层含义，核心要素是能指层面的故事性、可读性，以及是否有趣
    - 娱乐性: 剧情安排是陌生化的，具有强烈的娱乐性，读者能得到各种意想不到的体验
  - 虚拟性:
    - 故事发生在虚拟世界，但是和客观实在一样坚实可信，有其严整的逻辑性、历史背景...
  - 文学性:
    - 作品贴近现实主义精神。故事不一定在现实中真实发生，但它的投射具有真实生活的基础，让人觉得逻辑真实、细节真实、内涵真实。
    - 作品体现的不只是享乐主义，也会有一定的思想性，展现个人的世界观、人生观和价值观。
    - 文本以通俗为主但不排斥艺术性。俗中见雅，富有意味。
  - 重情绪:
    - 文本内在的情感充沛，使读者产生强烈的代入感
    - 注重满足读者的情绪价值，使读者在阅读中获得即时情感满足。
  - 画面感：
    - 画面感只是一种展现人物非言语(non-verbal)表现的手段，目的是追求深入人物内心的,并且足够真实的情绪表达。
    - 文字化转化：本质是对视听引发的情绪进行文字化转化，重点是“引发的情绪”，不是“视听”。要复现情绪产生的方式而不是画面本身，不要将文字劣化成视听的替代。
    - 精髓：在于文字描写视听场景时的文本叙述速度。并且考虑人物视角和心理描写的结合，遵从人物主观情绪的重点走。
    - 文字是经验的抽象，让文字再现视听语言唤起心理过程，体验到主体性之美
```
这些东西看上去很抽象，实际上没那么难写，你多读点论文就能写出来了，直接照抄然后自己稍微改改就好。
重点是你必须自己有清楚的认识，你想要什么样的效果，然后加自己想要的，无关的就不加。
我这个网文的文体特征强调重视情绪，文本符合现实主义精神，虽然通俗易懂但并不低俗。
这样是因为我对网文的文风也有一定的追求，我不太乐于看到那种比较烂俗的文字，玩梗没啥好玩的，看久了烦。
然后初学的话你未必能写出这么多的东西。
也不必一开始就想着写成我这样，你可以慢慢积累，平时没事抄点可能有用的句子，然后写的时候顺手用一下，感觉可以就留着，不好用就换别的。长期下来你也能积累很多有用的东西，写新文风也能套用很多东西。

**3. 叙述规则**  
这个叙述规则，就是文字结构。是为了实现基调、文体特征、杀八股的需求而制定的，具体的写作技巧
就比如说我对对话有特殊需求，他必须要独立成段，不带对话标签。不能比喻只能白描，不写眼神语气等等。
关于这些需求一般来说都是通用的，你写好了都能在别的文风里套用
我简单展示个示例，也不必照抄我的结构，你想加什么，想删什么都完全可以，这些不是一定必须的，你用的上才需要写。
```
- 叙事结构:
  - 文本组成部分:
    - 对话: 人物的话语
    - 非对话叙述
- 叙述句式:
    - 整体句式长短交错，叙述张弛有度，多用长自然段
- 书写规则
  - 对话规则：
    - 对话以自由直接引语主导
    - 不使用“某某说”这类词，而是用人物说话前/后的动作和叙述者我的独白，自然承接或引出对话
    - 连续的两人对话完全省略对话标签，通过一人一句对话的逻辑来区分说话人（2-5句）
    - 对话独立成段：对话与叙述分离。对话是独立的段落，不带任何对话标签
    - 符号规范: 对话用直角引号「」包裹。
  - 非对话叙述规则:
    - 文字是带有主观立场的诚实陈述，叙述者如实呈现记忆中的一切，没有任何隐瞒和虚构
    - 像写对话一样写叙述部分
  - 独白规则：
    - 用直接自由式转述人物内心没说出口的话（即独白）
    - 内心独白是无目标、无逻辑控制的自由联想
    - 独白混合到叙述流之中，给人思想不会中断的感觉
  - 表达方式:
    - 以现在时叙述历史，保证文本的流畅和代入感
    - 动作描写干脆利落，不修饰。不关注人物怎么说话怎么行动，直接呈现原始事实
    - 比喻规范：只白描，不比喻
    - 只写眼睛的物理状态和视线方向，不写眼睛的神态
  - 文字编排:
    - 学习中国散文大家，使用流水句式
    - 重视中国语文的特性，反对欧化中文
  - 语言体式：
    - 要有话说，方才说话，不要为文而文，没话找话
    - 要说我自己的话，所有文字本质都只是我在讲述
    - 文字要符合时代和地域特征。
    - 使用加工提炼过的生动活泼的口语书写，不用做作的文章腔调
    - 语言明白晓畅、细腻清晰、能准确表达意图。
  - 明确避免的情况:
    - 使用对话标签，如“他说”“她低声说”
    - 描写/解读说话者的语气神态、眼神
    - 揣测/解释非聚焦人物的心理/神态
    - 使用成语和四字词
```
建议包含“表达方式”“文字编排”“语言体式”及“禁止项”。  
“语言体式”中“语言明白晓畅、细腻清晰、能准确表达意图”是我在每个文风中都会强调的原则，能显著提升文字的干净度与表现力。  
“禁止项”用于强化前述规则，若发现模型未遵守某项要求，可在此补充强调。但若前面未提相关要求，禁止项可能反向成为提示，导致模型刻意违反。

四、案例研究——轻小说文风提示词的构建与实践  
我玩airp以来就特别想模仿入间人间的文风，所以我自己的期望目标就是入间人间的文字效果。
针对这一文体，我一开始就什么都不知道，不清楚要写什么，所以我立刻深入阅读分析文本。
我在早期文风创作中积累很少，也并没有认清自己要的到底是什么。
所以错误的选择了轻小说作为研究方向，不过这也不算走歪路，因为我认识到真正需要什么之后立刻就能把之前的经验全部迁移过来，做成了现在的自传体散文文风。
回归正题，我研究轻小说的成果就是：
  - 对话独立成段
  - 第一人称，非常强烈的主观限制视角
  - 有点意识流的意味
  - 对话多，独白多，轻松活泼不严肃，口语化
我就得到了一份比较简单的文风，这个文风属于我在创作三层架构前的文风，非常适合刚入坑文风的新人学习，按照这个样子写，堆料就好了。当堆到你自己都感觉不好改的时候，你就可以尝试去写一个厉害的框架，可以自创，也可以用我的。我认为在现有预设架构中，基调+文体特征+叙述规则，就已经是最优解，没有新东西了。


以下为早期撰写的轻小说文风示例（三层框架之前），适合新手参考：  
```
轻小说文风：
  - 基调：日式轻小说。以大量生动的对话、内心活动为主导。以读者平常使用的口语书写，不用严肃的文学化表达。
  - 读者：青少年（高中生至大学生）
  - 内心独白即旁白：正文就是聚焦人物记录的生活经历。所有的旁白本质都是聚焦人物在和读者对话。不存在一个客观的叙述者，这像是在写自传体散文
  - 绝大部分的旁白都属于聚焦人物思想和情感的抒发
  - 对话规则:
    - 对话与叙述分离：任何对话句都独立成段，常用多个人物连续的多轮对话段构成一个对话回合（两段以上），中间不穿插任何叙述。省掉所有引导性词句，例如：“xx说”、“xx想”、“xx道”
    - 符号规范: 对话用直角引号「」包裹，同时用直角双引号『』代替其他任何形式的引号
  - 意识流：呈现人物即时性、未经修饰的内心活动，并以丰富的联想为特点。并利用这种思维活动补充背景信息、塑造人物关系、揭示心理状态...
  - 外部信息的记叙与人物的内心活动融合在一起，无缝衔接。感知、行动与思维交织，形成自然的意识流。例如：“脚踏车载着拌嘴的两人，朝着桥梁离开。她们交情真好。还有会装熟。我明明直到上次都还和她们保持距离打交道，她们却已经将我当成颇熟的朋友。”
  - 确保正文中不含有任何学术报告、数据汇报等完全不会出现在口语中的内容，严格保证口语化
  - 口语示例(Only for reference): //使用平实自然的口语书写，如同人们生活中的日常闲聊。
    - 这是在我国中三年级时的春天所发生的事情。升上三年级后换了班级，我也刻意低调一点，结果不知道什么时候就被指派为图书委员了。正确来说应该是叫文化委员，但职务内容是负责顾图书室，所以我觉得说成图书委员也没关系。而我马上就被叫来执行我第一次的工作了。
```

五、其他文风方法讲解  
**1. 人格化**  
（这种方式很难写，而且没有显著优于规则化文风，所以没有很推荐）
接下来，我将介绍另一种撰写文风提示词的方法。之前提到的方法比较指令化，略显机械化，只是向模型下达指令。而文风还有另一种优秀的表现形式：人格化。
具体来说，人格化写法包括：
- 参考原作者的口吻描述规则。
- 通过阅读访谈、作者自述等资料，补充具体的写作规则。
这种方法需要足够的积累，否则很难模仿作家的口吻。我最初也尝试过这种方法，虽然尚未精进，但效果确实非常有趣。
ryou曾向我详细讲解过这种方法的写法。他最初是认真研究帕拉尼克的作品风格，并从《搏击俱乐部》中汲取灵感。《搏击俱乐部》中提到了八条规则，于是他尝试用作家的口吻撰写文风规则。最终的效果非常出色，后续只需根据作者的实际风格调整口吻即可。
这种人格化的思路对抑制八股也有很大帮助。关键在于识别作者和作品的特点，抓住核心，并进行模仿，用作者的典型意象替代一般化的意象表达。然后将这些元素组合起来，形成一个看似莫名其妙、实则蕴含写作规则的文本。这同时合并了规则和语料。
同时要注意，“作品的口吻”比“作者本人的话”要更重要，我们可以采纳作者原话，但是不能生搬硬套，像告示板一样四处张贴，人格化本身风格和逻辑要统一。
当然，这种方法可能会与预设的人格化产生冲突，因此可能需要补充一些规则来修复问题。
对于外国作品，需要注意：翻译文本会受到译者的影响，因此要分辨你捕捉到的是译者的风格还是作者的风格。这一点需要自行判断。原语种关于作者的语料通常更丰富，因此模型应该能更好地捕捉作者的特点。

**2. 示例流**
（Gemini3.0非常吃示例，所以可以写一个简单的规则搭配示例，示例大概四百token就够，整体加起来不超过一千token会很好）
eula走的是示例流，写下了通用的散文文风，只要换文风示例就能吃到不同风味的文风，这个方法是真的挺好的。
我自己是因为写文风token偏大，是零样本写法，把开场白当作文风示例用。这样写也没有问题。
对于示例流而言，框架要是偏大众的，所以完全能用你在小学初中高中学到的分析方法，没有思路也可以返回到课文里看看教辅资料怎么赏析这一段文字的，或者想想你最喜欢的作品是怎么让你感动的。
```
  - 文体定义:
    - 「闲谈」不是做文章，呈现的是故事生活中的视觉语⾔
    - 「散文」不存在典型，结构松散，追求内在的意境。
    - 「小说」是载体，不论写人、记事、状物都产⽣具体的实感，以区别于其他的书⾯语⾔。
  - 文体特征:
    - 语言准确，畅通，平易，雅致。语句短长，话说得完说不完，深思熟虑临时起意，有理没理皆可。
    - 自然景物、市井风情、家庭琐事、婚姻爱情，无所不写，把平平淡淡的事说得很有情致。
    - 关注的仅是⽣活的⼀处⾓落，⼀个⽚段。即使有重⼤题材，也会把它⼤事化⼩。
    - 不过分刻画人物，不去描写/解读说话者的语气神态、眼神，也不揣测/解释人物的心理/神态。
```
但是示例流对于示例本身有一定要求，需要包含一段完整的场景，而非简单的描述，因为我们要ai写的是文章。以闲谈式的第一和第三人称为例。
```
      我在好几篇小说中都提到过一座废弃的古园，实际就是地坛。
      许多年前旅游业还没有开展，园子荒芜冷落得如同一片野地，很少被人记起。
      地坛离我家很近。或者说我家离地坛很近。总之，只好认为这是缘分。地坛在我出生前四百多年就座落在那儿了，而自从我的祖母年轻时带着我父亲来到北京，就一直住在离它不远的地方——五十多年间搬过几次家，可搬来搬去总是在它周围，而且是越撤离它越近了。我常觉得这中间有着宿命的味道：仿佛这古园就是为了等我，而历尽沧桑在那儿等待了四百多年。
      它等待我出生，然后又等待我活到最狂妄的年龄上忽地残废了双腿。四百多年里，它一面剥蚀了古殿檐头浮夸的琉璃，淡褪了门壁上炫耀的朱红，坍记了一段段高墙又散落了玉砌雕栏，祭坛四周的老柏树愈见苍幽，到处的野草荒藤也都茂盛得自在坦荡。
      这时候想必我是该来了。十五年前的一个下午，我摇着轮椅进入园中，它为一个失魂落魄的人把一切都准备好了。那时，太阳循着亘古不变的路途正越来越大，也越红。在满园弥漫的沉静光芒中，一个人更容易看到时间，并看见自己的身影。
      自从那个下午我无意中进了这园子，就再没长久地离开过它。
      我一下子就理解了它的意图。正如我在一篇小说中所说的：“在人口密聚的城市里，有这样一个宁静的去处，像是上帝的苦心安排。”
```
第一人称采用的史铁生的《我与地坛》，这里的闲谈展示的就是作者的生活还有思绪，因为eula认为第一人称更需要体现的是user。
第三人称他选用的是汪曾祺的《受戒》，这里的闲谈就是在介绍故事，他认为第三人称的优势在于能更自由地讲述世界和故事。同时这也是一个精选的对话片段，可以看到原文的对话和段落都很简洁，没有多余的解释。
```
    舅舅按照约定的日期又回了家，带了一件他自己穿的和尚领的短衫，叫明子娘改小一点，给明子穿上。明子穿了这件和尚短衫，下身还是在家穿的紫花裤子，赤脚穿了一双新布鞋，跟他爹、他娘磕了一个头，就随舅舅走了。
    他上学时起了个学名，叫明海。舅舅说，不用改了。于是“明海”就从学名变成了法名。
    过了一个湖。好大一个湖！穿过一个县城。县城真热闹：官盐店，税务局，肉铺里挂着成爿的猪肉，一个驴子在磨芝麻，满街都是小磨香油的香味，布店，卖茉莉粉、梳头油的什么斋，卖绒花的，卖丝线的，打把式卖膏药的，吹糖人的，耍蛇的……他什么都想看看。舅舅一劲地推他：“快走！快走！”
    到了一个河边，有一只船在等着他们。船上有一个五十来岁的瘦长瘦长的大伯，船头蹲着一个跟明子差不多大的女孩子，在剥一个莲蓬吃。明子和舅舅坐到舱里，船就开了。
    明子听见有人跟他说话，是那个女孩子。
    “是你要到荸荠庵当和尚吗？”
    明子点点头。
    “当和尚要烧戒疤！你不怕？”
    明子不知道怎么回答，就含含糊糊地摇了摇头。
    “你叫什么？”
    “明海。”
    “在家的时候？”
    “叫明子。”
    “明子！我叫小英子！我们是邻居。我家挨着荸荠庵。——给你！”
    小英子把吃剩的半个莲蓬扔给明海，小明子就剥开莲蓬壳，一颗一颗吃起来。
```
示例也是把双刃剑，我们想象不出来童话故事里会有三体人，也不会让三体人出现在那里，但是ai会——示例对文字的基调有很强的影响。如果闲谈式的示例塞的是色情小说的内容的话，可能是一个嫖客在和你说他的光辉事迹。
